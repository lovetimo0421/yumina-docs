# 截图清单

把截好的图放到这个文件夹里，文件名要和下面列的一致。格式推荐 PNG。

## 新手指南（01-beginner-guide.md）

| 文件名 | 截什么 |
|--------|--------|
| `create-world.png` | 点 Create 按钮后的模板选择页（Create New），几个模板卡片并排 |
| `editor-overview.png` | 编辑器全貌，左侧导航栏完整展示 |
| `first-message.png` | First Message 编辑区域，带一段示例内容 |
| `lorebook-list.png` | Lorebook 区域，左侧词条列表，右侧编辑面板 |
| `lorebook-entry-detail.png` | 一个词条的编辑面板，能看到 Role、Always Send、Keywords、Send As、Position |
| `variables.png` | Variables 区域，几个不同类型的变量 |
| `behaviors.png` | Behaviors 区域，一条行为的编辑面板（WHEN / ONLY IF / DO） |
| `message-renderer.png` | Message Renderer 区域，能看到 Default/Custom TSX 两个选项 |
| `audio.png` | Audio 区域，音频轨道列表 |
| `assets.png` | Assets 区域，已上传的图片缩略图 |
| `overview.png` | Overview 区域，Cover Image、Description、Tags、Publish 按钮 |
