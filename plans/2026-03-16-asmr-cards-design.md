# ASMR Cards Design — asmr.one 作品改编

## 概述

从 asmr.one（DLsite 音声作品库）选取 4 部高人气作品，生成适配 Yumina 的 WorldDefinition v13 卡片。2 SFW + 2 NSFW，各有定制 messageRenderer。

## 数据源

asmr.one REST API:
- 作品详情: `https://api.asmr.one/api/work/{id}`
- 文件列表: `https://api.asmr.one/api/tracks/{id}`
- 封面图: `https://api.asmr.one/api/cover/{id}.jpg?type=main`
- DLsite 页面: `https://www.dlsite.com/maniax/work/=/product_id/RJ{id}.html`

## 通用要求

1. **全部中文** — 角色名、变量名、规则名、lorebook entry name、所有用户可见文字
2. **messageRenderer** — 每张卡定制 TSX 前端，风格各异
3. **封面图** — 使用 asmr.one 封面 URL 或作品内包装图
4. **ASMR 特色** — 所有卡的 system prompt 都要强调 ASMR 写作风格：环绕音效描写、耳语距离感、触觉/听觉细节、亲密空间感
5. **输出目录** — `output/`
